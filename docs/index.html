Built to survive

Everything you need for
reliable live streaming

Every edge case handled — from Samsung's battery killer to 3 AM network blips.

HLS Live Streaming

ExoPlayer (Media3) with full HLS support and OkHttp networking for any live audio stream URL.

Foreground Service

Persistent media-style notification with Play/Stop controls. Android cannot kill it silently.

Auto-start on Boot

BootReceiver intercepts BOOT_COMPLETED and restarts your stream automatically after every reboot.

Dual Watchdogs

AlarmManager (2 min, Doze-resistant) + WorkManager (15 min) restart the stream after any crash.

Network Recovery

Detects when connectivity is restored and reconnects instantly — no manual action required.

Headphone Aware

Pauses automatically when headphones or Bluetooth audio devices are disconnected.

Battery Optimised

Doze-resistant setExactAndAllowWhileIdle() alarms. Handles Samsung and other aggressive OEMs.

Per-app Volume

Independent 1–100% volume. Plays alongside GPS, music, or calls without claiming audio focus.

Configuration

Settings Reference

Tap Settings in the app to configure all options.

Setting Default Description
Stream Name Live Stream Display name shown in the notification bar.
Stream URL radiojar default Full HTTP(S) URL of the audio stream to play.
Auto-start on Boot Enabled Starts the stream automatically when the device reboots.
Auto-relaunch if Stopped Enabled Watchdogs restart the stream after crashes, network loss, or unexpected stops.
Reconnect Delay 5 s Base delay before the first reconnect attempt (uses exponential backoff).
Max Retries −1 (∞) Maximum reconnect attempts. −1 retries forever.
Playback Volume 100% App-level volume (1–100%), independent of the system volume.
